# Life-Helper

A Web app for life

## Getting Started

### Installing

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn dev
```

### Lints and fixes files

```
yarn lint
yarn lint:fix
```

### Run your unit tests

```
yarn test
```
